Сервис для обмена данными с часами серии reTRONIX

1. Сконфигурируйте сервис с помощью скрипта "Конфигурация сервиса.bat". Выражения в квадратных скобках задают значения настроек по-умолчанию. Для их применения нажмите "Enter" без ввода данных.
2. Установите сервис. Для установки сервиса требуются права администратора, они автоматически запросятся при установке. Для установки сервиса запустите скрипт "Установить сервис reTRONIX.os.bat".
3. Данные передаются на устройство с указанным именем по порту 5000.

Для внесения изменений в настройки сервиса сначала запустите скрипт "Остановка сервиса вручную.bat". После этого внесите изменения в настройки с помощью скрипта "Конфигурация сервиса.bat". После завершения скрипта запустите скрипт "Запуск сервиса вручную.bat".

Скрипт "Удалить сервис reTRONIX.os.bat" удалит сервис из системы.
